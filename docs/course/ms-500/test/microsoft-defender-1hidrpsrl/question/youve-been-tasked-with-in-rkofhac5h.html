Your organization has a Microsoft 365 tenant with Microsoft 365 E5 licenses assigned to each active user.

Your organization is currently using Microsoft Defender for Endpoint. You've deployed Defender for Endpoint to each of your computers using Microsoft Intune.

Your manager learns that Microsoft Defender for Office 365 and Microsoft Defender for Endpoint can be integrated. He loves the idea.

He has asked you to integrate Microsoft Defender for Office 365 and Microsoft Defender for Endpoint.

Where do you configure the integration?

Answer Explanation

1. Go to Microsoft 365 Defender admin center > Explorer > MDE Settings. Set Connect to Defender for Endpoint to On.
